① Core formulas (build-up)

⚑ Mean:  x̄ = Σx / N ⚑ Sum of Squares:  SS = Σ(x − x̄)² ⚑ Variance:  SD² = SS / N ⚑ Std Dev:  SD = √(SS / N) SP = Σ(x − x̄)(y − ȳ) COV = SP / N r = COV / (SDx · SDy)

Do SS & SD for X and Y separately.

② Calculation order (the table)

One table, columns:

X · Y · (x−x̄) · (y−ȳ) · (x−x̄)² · (y−ȳ)² · (x−x̄)(y−ȳ)

Sum the last 3 cols → SSx, SSy, SP. Then SDx, SDy, COV, r.

Round 4 dp during, 2 dp final.

③ Describe a correlation

  • Direction + / − / none
  • Shape linear / curvilinear
  • Strength weak/mod/strong (subjective)
  • Magnitude = r value (−1 to +1; objective)

In words = 3 terms: strength + direction + shape (e.g. "strong, negative, linear").

Also use Spearman if I/R data is curvilinear or heteroscedastic (ranks the scores, pulls outliers in).

⑥ Factors affecting r

Does NOT affect r: sample size (N cancels).

  1. Real strength of relationship
  2. Sampling error (↑ or ↓ r)
  3. 3rd variable (corr ≠ cause)
  4. Heterogeneous sample (subgroups)
  5. Restricted/truncated range
  6. Non-linearity → r underestimates
  7. Heteroscedasticity → SD↑ so r↓

⑦ Regression formulas (provided)

Slope:  by = r · (SDy / SDx) Intercept:  ay = ȳ − by(x̄) Line:  Y′ = ay + by(x)
bx = r · (SDx / SDy) ax = x̄ − bx(ȳ) X′ = ax + bx(y)

⑧ Regression pipeline

  1. Scatterplot + mean & SD each var
  2. Pearson r
  3. by = r·SDy/SDx
  4. ay = ȳ − by(x̄)
  5. Y′ = ay + by(x) → plug in X
  6. Plot via (0, ay) and (x̄, ȳ)

X′: same steps, swap X & Y.

⑨ Predicting — KEY RULES

  • Predictor = X (predict from). Criterion = Y (predicted).
  • NO extrapolation: if the value is outside the observed range → can't predict → write NA.
  • Plug into Y′ (or X′); show formula → numbers → answer.

⑩ Two lines: Y′ & X′

  • Cross at the means (x̄, ȳ) (if not → error).
  • Rotate around (x̄, ȳ) as r changes.
  • Small angle = high r. r = ±1 → lines overlap. r = 0 → perpendicular (90°).

⑪ r, b, a relationship

  • r = 0 → b = 0; Y′ flat at ay = ȳ; X′ vertical at ax = x̄.
  • r ↑ → |b| ↑ (steeper).
  • b + → a below mean (toward origin); b − → a above mean.

⑫ Std error of estimate

SD(Y−Y′) = SDy · √(1 − r²) = √( Σ(Y−Y′)² / N )
  • r ≠ 0 → smaller than SDy
  • r = 0 → equals SDy
  • r = ±1 → = 0 (no error)

⑬ Variability (r²)

Total (Y−ȳ) = Explained (Y′−ȳ) + Unexpl. (Y−Y′) SST = SSR + SSE
  • r² = SSR/SST = proportion explained
  • 1 − r² = SSE/SST = proportion unexplained

e.g. r=.95 → r²=.90 → 90% explained.

⑭ Key concepts (1-liners)

  • Least squares: best line minimizes Σ(Y−Y′)².
  • Heteroscedasticity (links both topics): Y-spread changes across X (fan). ↓ r, flattens slope → use Spearman.
  • Curvilinear: Pearson underestimates → Spearman.
  • Restricted range: can't generalize beyond sampled range.
  • Corr ≠ cause: need an experiment.
